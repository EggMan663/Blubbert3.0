Blubbert 3.0 is a Discord bot that can learn new phrases and recall them randomly when you mention its name. It also features a quotebook that allows you to retrieve random quotes from any person or specifically from a chosen individual. You can add quotes to people's quotebooks and introduce new people to the quotebook.

Features
Teach Blubbert new phrases
Retrieve random quotes from the quotebook
Add new quotes to the quotebook
Add new people to the quotebook

Commands
b!teach: Teach Blubbert a new phrase. Follow the prompts to provide a question and an answer for the bot.
b!qb: Retrieve a random quote from the quotebook. If an author's name is provided, a random quote from that author will be displayed. If no author's name is provided, a random quote from any author will be displayed.
b!qb add [author_name] [quote]: Add a new quote to the quotebook.

Setup
Clone this repository.
Install the required dependencies with pip install -r requirements.txt.
Create a .env file in the root directory and add your Discord bot token like so: TOKEN=your-bot-token.
Run the bot with python bot.py.
